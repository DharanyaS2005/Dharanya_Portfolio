EXCEL TRACKER

This project automates the process of checking Google Sheets for missing data entries and sending email notifications to the respective users. Utilizing Django for the web interface, Google Sheets API for data retrieval, and SMTP for email dispatch, it ensures timely and efficient communication regarding incomplete information. The application also incorporates scheduling to perform these checks and notifications at regular intervals, enhancing data management and user compliance.
